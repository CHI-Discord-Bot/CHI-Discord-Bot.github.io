Documentation - Management Commands

Commands and functions

;addrole

Use this command to add a role to an user. Use this command like this: ;addrole [user] [role]

;remrole

Use this command to remove a role from an user. Use this command like this: ;remrole [user] [role]

;kick

Use this command to kick an user. Use this command like this: ;kick [user] [reason]

;ban

Use this command to ban an user. Use this command like this: ;ban [user] [reason]

;unban

Use this command to unban an user. Use this command like this: ;unban [user id]

;nickname

Use this command to change the nickname of one specific user. Use this command like this: ;nickname [user] [nickname]. If you don't give any nickname, the nickname gets reset.

;nicknames

Use this command to change the nicknames of all users in your server. Use this command like this: ;nicknames [nickname]. If you don't give any nickname, the nicknames of all users on your server gets reset.

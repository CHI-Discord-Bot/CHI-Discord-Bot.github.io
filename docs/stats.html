Documentation - Stats Commands

Commands and functions

;vcstats

Returns the voice channel stats of an user. If no user is given, the bot returns the voice channel stats for the message author.

;vctop

Returns the top 10 voice channel activity list of the server. If nothing is given behind the command, the bot returns the top list of all time. If an integer is given, it returns the top list of the last [your integer] days.

;vctopglobal

The same as in ;vctop but with one difference. This command returns the top 10 voice channel activity list of all users in all servers the bot has access to.

;vctopserver

This command returns the top 10 voice activity list of all the servers the bot is in. The bot summarized all user activity in each server and compares it to each other. You can also give an integer in this command to only get the stats of the last few days.

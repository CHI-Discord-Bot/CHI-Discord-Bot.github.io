Documentation - Voice Channel Commands

Commands and functions

;join

The bot joins the voice channel you are currently in.

;leave

The bot leaves the voice channel you are currently in.

;vcmute

Server mutes all members in the voice channel you are currently in.

;vcunmute

Server unmutes all members in the voice channel you are currently in.

;play

Use this command if you want the bot to play some music in voice channel. To use it, you have to give a YouTube-Link.

e.g.: ;play www.randomyoutubelink.com

;playl

When you use this command the bot will play the last song that was given to ;play.

;pause

Pausing the currently playing music.

;creator

Resuming the currently paused music.
